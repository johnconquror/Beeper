  1. Go to developer.spotify.com/dashboard and sign in.
  2. Click Create app. Name it anything (e.g. "Beeper").
  3. Set Redirect URI to exactly:
  4. Check Web API and Web Playback SDK, then save.

⚠ You opened this from a local file. Spotify won't accept file:// redirect URIs. Run a tiny local server in this folder, e.g.
